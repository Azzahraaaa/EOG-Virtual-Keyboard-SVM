# Penerapan Support Vector Machine (SVM) pada Klasifikasi Sinyal EOG untuk Virtual Keyboard
# Project ini merupakan implementasi metode Support Vector Machine (SVM) untuk mengklasifikasikan sinyal Electrooculography (EOG) pada sistem virtual keyboard berbasis gerakan mata.
Sistem ini dirancang untuk membantu pengguna dengan keterbatasan motorik dalam melakukan input teks menggunakan pergerakan mata.
Fokus klasifikasi:
- Lirikan mata ke atas
- Kedipan mata
Tahapan penelitian:
1. Akuisisi sinyal EOG
2. Preprocessing (pengurangan noise dan normalisasi)
3. Ekstraksi fitur sinyal
4. Klasifikasi menggunakan SVM dengan kernel RBF
5. Evaluasi performa (akurasi, error rate, response time)
Hasil
Penerapan SVM menunjukkan:
- Peningkatan akurasi sebesar 17%
- Penurunan error rate sebesar 18.5%
- Waktu respons sistem lebih cepat sebesar 21%
Teknologi yang Digunakan:
- Python
- Scikit-learn
- NumPy
- Pandas
- Matplotlib
- Signal Processing
